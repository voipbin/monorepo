.. _domain-struct-domain-domain:

Domain
------

.. code::

    {
        "id": "<string>",
        "name": "<string>",
        "detail": "<string>",
        "domain_name": "<string>",
        "tm_create": "<string>",
        "tm_update": "<string>",
        "tm_delete": "<string>"
    },

* *id*: Domain's id.
* *name*: Domain's name.
* *detail*: Domain's detail description.
* *domain_name*: Target domain name. Must include '.sip.voipbin.net'.
