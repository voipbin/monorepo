Recording

{
    "id": "<string>",
    "type": "<string>",
    "reference_id": "<string>",
    "status": "<string>",
    "format": "<string>",
    "tm_start": "<string>",
    "tm_end": "<string>",
    "tm_create": "<string>",
    "tm_update": "<string>",
    "tm_delete": "<string>"
}

  • id: Recording’s ID.

  • type: Recording’s type. See detail here.

  • reference_id: Reference’s ID. It shows call/conference’s ID.

  • status: Recording status. See detail here.

  • format: Recording file format. See detail here.
