Tag

Overview

The tag is used for queue and agent.

Tag

Tag

{
    "id": "<string>",
    "name": "<string>",
    "detail": "<string>",
    "tm_create": "<string>",
    "tm_update": "<string>",
    "tm_delete": "<string>"
}

  • id: Tag’s id.

  • name: Tag’s name.

  • detail: Tag’s detail description.
