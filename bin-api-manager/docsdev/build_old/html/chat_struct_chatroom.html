Chatroom

{
    "id": "<string>",
    "customer_id": "<string>",
    "type": "<string>",
    "chat_id": "<string>",
    "owner_id": "<string>",
    "participant_ids": [
        "<string>",
        ...
    ],
    "name": "<string>",
    "detail": "<string>",
    "tm_create": "<string>",
    "tm_update": "<string>",
    "tm_delete": "<string>"
}

  • id: Chat’s ID.

  • customer_id: Customer’s ID.

  • type: Chat’s type. See detail here.

  • chat_id: Master chat’s ID.

  • owner_id: Owner’s ID.

  • participant_ids: list of participate ids.

  • name: Chat’s name.

  • detail: Chat’s detail.
