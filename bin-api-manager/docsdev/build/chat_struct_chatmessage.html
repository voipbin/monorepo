Chatmessage

Chatmessage

{
    "id": "<string>",
    "customer_id": "<string>",
    "chat_id": "<string>",
    "source": {
        ...
    },
    "type": "<string>",
    "text": "<string>",
    "medias": [
        ...
    ],
    "tm_create": "<string>",
    "tm_update": "<string>",
    "tm_delete": "<string>",
}

  • id: Chat’s ID.

  • customer_id: Customer’s ID.

  • chat_id: Master chat’s ID.

  • source: Source info. See detail here.

  • type: Chatmessage’s type. See detail here.

  • text: Text message.

  • medias: List of medias.

    "id": "2b4acb7b-f1ba-43c5-ae43-0435a07d55ea",

        "type": "agent",
