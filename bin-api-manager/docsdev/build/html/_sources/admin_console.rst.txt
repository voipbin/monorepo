*******************
Admin Console Guide
*******************

.. note:: **AI Context**

   This section documents the **admin.voipbin.net** web console, the
   browser-based UI for managing a VoIPBin account (flows, numbers, agents,
   AI assistants, queues, and more). It is written for day-to-day console
   users, not for developers integrating against the REST API. For the API
   itself, see the :ref:`Quickstart <quickstart-main>` guide and the other
   sections of this site.

The admin console is organized as a left-hand sidebar with six top-level
groups (Voice, Messaging, AI Services, Timeline, Platform, Operations), a
top bar with account/billing status, and a main content area that shows a
list, detail, or editor view depending on what you selected. Every
resource follows the same list -> detail -> create pattern, so once you
learn one you can find your way around the rest.

Use the search box at the top of any page in this guide (or the sidebar
search on docs.voipbin.net) to jump straight to a topic, for example
"flow", "number", or "assistant".
